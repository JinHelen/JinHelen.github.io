图书推荐系统（零）之Django创建

· 2019-04-20 · # 推荐系统

1 配置Django

1.1 创建项目

django-admin startproject Book

1.2 创建APP

python manage.py startapp index
python manage.py startapp user
python manage.py startapp rank

1.3 创建后台管理员（这里的管理员为admin/admin）

python manage.py createsuperuser

1.4 创建数据库

写上面这两句之后，输入密码，进入数据库

create database book

1.5 在django项目配置mysql

（1）在项目文件下的settings.py文件里修改：

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'book',
        'USER':'root',#这里改成自己的数据库名字
        'PASSWORD':'123456',#这里改成自己数据库的密码
        'HOST':'127.0.0.1',
        'PORT':'3306',
    }
}

（2）在项目文件下的__init__.py文件里添加：

import pymysql
pymysql.install_as_MySQLdb()

1.6 在项目文件下的settings.py文件里修改app：

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'index',
    'user',
    'rank',
]
