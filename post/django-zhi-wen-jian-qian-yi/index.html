Django之文件迁移

2019-04-19

那些年遇到的django的坑

安装依赖包

requirements.txt 内容如下：

Django==2.1.0
requests==2.21.0
PyMySQL==0.9.2
BeautifulSoup4==4.6.3
setuptools==39.1.0

在终端执行

pip install -r requirements.txt

启动django

（1）创建项目

django-admin startproject BookRec

（2）创建APP

python manage.py startapp index
python manage.py startapp user
python manage.py startapp rank

（3）创建数据库bookrec

写上面这两句之后，输入密码，进入数据库

create database bookrec

（4）在django项目配置mysql

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'bookrec',
        'USER':'root',#这里改成自己的数据库名字
        'PASSWORD':'123456',#这里改成自己数据库的密码
        'HOST':'127.0.0.1',
        'PORT':'3306',
    }
}

（5）执行迁移，数据库同步

python manage.py makemigrations
python manage.py migrate

（6）将数据写入数据库 运行z_data/to_mysql里的代码，即可

（7）创建后台管理员（这里的管理员为admin/admin）

python manage.py createsuperuser

（8）启动服务

python manage.py runserver
