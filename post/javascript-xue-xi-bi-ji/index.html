JavaScript学习笔记

· 2019-04-23 ·

1、什么是JavaScript？

JavaScript是一种运行在浏览器中的解释型的编程语言。 它可用于网页特效、表单验证、响应事件等等。

2、基础知识

2.1 存放位置

（1）JavaScript可以嵌在网页的任何地方，习惯放在<head>中。 （2）把JavaScript代码放到一个单独的.js文件，然后在HTML中通过<script src="..."></script>引入这个文件：

2.2 基本语法

JavaScript的语法和Java语言类似，每个语句以;结束，语句块用{...}。

if (2 > 1) {
    x = 1;
    y = 2;
    z = 3;
}

注释：采用//或/*...*/

2.3 数据类型

（1）数组：用[]表示

[1, 2, 3.14, 'Hello', null, true];
new Array(1, 2, 3); // 创建了数组[1, 2, 3]

（2）对象：键值对

var person = {
    name: 'Bob',
    age: 20,
    tags: ['js', 'web', 'mobile'],
    city: 'Beijing',
};

person.name; // 'Bob'
